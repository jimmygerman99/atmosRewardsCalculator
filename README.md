# Atmos Rewards Calculator

A web calculator for estimating **Atmos Miles** and **Status Points** earned through the Atmos Rewards loyalty program (Alaska Airlines + Hawaiian Airlines).

## Features

- Flight earnings — distance, spend, or segment method (2026)
- Credit card earnings — Atmos Ascent, Summit, Alaska Business, Hawaiian Barclays
- Partner earnings — hotels, car rentals, cruises, Lyft, shopping portal, SAF, GCI
- Elite status progress bar with milestone tracking (Silver → Gold → Platinum → Titanium)

## Stack

React + TypeScript + Vite + Tailwind CSS
